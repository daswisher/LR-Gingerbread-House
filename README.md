## Introduction
This is the Labs project for the Gingerbread House competition.

### Features
* Interactive gingerbread house
* IoT
* GPIO using a Raspberry Pi 2
* Entirely written in Python

## Getting Started

### Prerequisites
* Python 2.7
* Raspberry Pi 2/3
* Raspbian
* Internet connection
* Sonar sensors, LEDs, and servos
